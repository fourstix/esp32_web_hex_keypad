# esp32_web_hex_keypad
ESP32 web-based Hexadecimal Keypad

## Hardware
* Controller: ESP32DEV-C
* Relay Module: Sainsmart 16 Relay Module https://www.sainsmart.com/products/16-channel-12v-relay-module
